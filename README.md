# 🛡️ SafeScaner v4.1.1
**A lightweight and transparent batch script for primary Windows security audit and threat detection.**

---

## 🇷🇺 Описание (Russian)
Данный скрипт предназначен для выявления следов вредоносной активности (стиллеров, майнеров) в скриптовых файлах и системных процессах. 

### 🎯 Почему SafeScaner?
*   **100% Открытый код:** Вы можете проверить каждую строку в Блокноте.
*   **Прозрачность:** Никаких скрытых функций или выхода в интернет.
*   **Контроль:** Удаление файлов только с вашего подтверждения.

---

## 🇺🇸 Description (English)
A primary security audit tool to identify malicious traces within script files and system processes.

### 🎯 Why SafeScaner?
*   **100% Open Source:** Verify every line in Notepad.
*   **Transparency:** No hidden functions or network connections.
*   **Full Control:** Files are deleted only with user confirmation.

---

## 🛠️ Как использовать / How to use
1. Скачайте `SafeScaner.bat`.
2. Запустите от имени Администратора.
3. Дождитесь окончания сканирования и проверьте папку `/logs`.
