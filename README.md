🛡️ SafeScaner v4.1.2 — Stable Release

SafeScaner — это мощный легковесный инструмент для сигнатурного анализа файлов и поиска скрытых угроз (стиллеров, майнеров, вредоносных скриптов) в вашей системе.
⚠️ Важное объявление: Откат с версии 5.0
Версия SafeScaner v5.0 была официально отозвана и удалена из репозитория.
Причина: В ходе массового тестирования была обнаружена критическая ошибка в ядре сканера, приводившая к «циклическому самодетекту» и нестабильной работе при анализе системных директорий.
Мы приняли решение вернуться к проверенной архитектуре 4-й ветки, внедрив в неё все необходимые исправления. Версия 4.1.2 — это текущий стандарт безопасности проекта.
🔥 Что нового в v4.1.2?
В этом обновлении мы объединили стабильность старых билдов и новые возможности анализа:
🚀 Расширенный охват: Теперь сканер проверяет файлы форматов .exe, .bat, .py, .ps1, .vbs, .cmd и .txt.
🔍 Улучшенные базы: Обновлены сигнатуры для поиска современных стилеров (детект обращения к Login Data, Cookies, Web Data).
⚡ Живой мониторинг: Отображение процесса сканирования в консоли в реальном времени.
📝 Умные логи: Автоматическая генерация отчета scan_report.txt и его открытие сразу после завершения работы.
🛡️ Anti-Self-Detect: Полностью переработана логика исключений — сканер больше не детектит собственный код.
🚀 Как использовать
Скачайте файл SafeScan_v4.1.2.zip-.
Запустите его (желательно от имени администратора для доступа к системным папкам).
Дождитесь окончания сканирования.
Выберите действие: Удалить найденные угрозы или Сохранить их для ручного анализа.
Изучите открывшийся отчет.
